Assignment-2 Interactive Visualization

This assignment is to explore the data set of 11 years Men US Open Matches. I have used D3.js to produce an interactive visualization. Also, write up to describe the story is submitted with this assignment.

Viz -1

This visualization is a simple Scatterplot using the match performance variables of all the winners.
X-axis-> is the Winning Player's Average Winner points in all played matches.
Y-axis-> is the Winning Player's Average Error points in all played matches.
Circle/Bubble-> represents each player who won any match in all played matches.
Size of the circle-> represents the Winning Player's Average return points in all played matches.

Change X-Axis»

  • The above scatterplot is Zoomable and so it can be zoomed and observed precisely.
  • Each bubble has a tool tip where it shows the complete details of it. Each bubble's tool tip shows Player's name, his Avg Winner Points, his Avg Error Point, and his Avg Return Percentage
  • On clicking the above blue button Change X-Axis, the X-Axis will change will show the Avg Return Values and this thus the plot will be Avg Error vs Avg Return.
  • The scale will change automatically on click of the button

Viz -2

This visualization from Tableau shows the trend lines of the variables we used in Viz-1. It clearly shows that there is a linear relationship among Winner Points and Error Points.
A linear trend model is computed for average of Error1 given average of Winner1. The model may be significant at p <= 0.05. This is the same visualization as Viz-1. we are using the same parameters here, its just here we are focussing only on top 20 players
